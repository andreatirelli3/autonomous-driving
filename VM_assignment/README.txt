Struttura del progetto

I report sono collocati nella cartella 'doc/' suddivisi per:

- assignment1/
- assignment2/

La struttura del progetto è la seguente

- common/ -> funzioni generali per le simulazioni
- controllers/ -> funzioni e simulazioni dei controllori di un veicolo
	+ steering/ -> controllori dello sterzo (pole placement e lqr)
	+ velocity/ -> controllori di velocità (PI)
- doc/ -> documenti e report per i vari assignment
	+ assignment1/ -> documenti assignment 1
	+ assignment2/ -> documenti assignment 2
- models/ -> modelli longitudinali e laterali di un veicolo (assignment1)
- static/ -> cartella dei file statici (utili per i file .md)
